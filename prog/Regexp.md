## Regexp

```py
import re

strings_to_match = [
    "toto",
    "foo",
    "bar",
    "pouet1",
    "pouet2",
    "root",
    "toor"
]

def regex_v1():
    reg = re.compile(r"pouet")
    for s in strings_to_match:
       if reg.match(s):
           print(f"Match: {s}")

def regex_v2():
    for s in strings_to_match:
       if re.match(r"pouet", s):
           print(f"Match: {s}")
```

## Compile : n operations vs 2n avec match
