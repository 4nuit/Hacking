## Tools

- [astring](https://github.com/davidbonnet/astring) - ESTree-to-JS **code generator** (AST → source), not a parser
- https://beautiful-soup-4.readthedocs.io/en/latest/
- [tree-sitter](https://tree-sitter.github.io/tree-sitter/) - incremental parser generator producing a concrete syntax tree, used by most modern editors/static-analysis tools (many ready-made grammars)
- [ANTLR](https://www.antlr.org/) - parser generator (LL(*)), widely used for building custom language parsers from a grammar file
